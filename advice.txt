0. Использование Localtunnel (остальные сервисы не работают в РФ или их работа затруднена)
npm install -g localtunnel
lt --port 3000 (вводим номер порта, который нужно пробросить)

Бесплатный сервис, требуется посомтреть свой IP и направлять его другим пользователям, чтобы они смогли зайти, скромное быстродействие

1. Заходим на свой сервер TimeWeb (к примеру)
Установить nginx на сервере
sudo apt update
sudo apt install nginx

Создать файл с настройками
nano /etc/nginx/sites-available/api

Содердание файла (главное корректно указать порты, на котором работает наша программа):
server {
    listen 80;
    server_name _.__.___.___; (IP сервера)

    # Увеличьте таймауты для долгих запросов
    proxy_connect_timeout 600;
    proxy_send_timeout 600;
    proxy_read_timeout 600;
    send_timeout 600;

    # Frontend - React/Vue/Angular etc (port 3000)
    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    # Backend API (port 8000)
    location /api/ {
        proxy_pass http://127.0.0.1:8000/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;

        # Дополнительные настройки для долгих запросов
        proxy_buffering off;
        client_max_body_size 0;
        proxy_connect_timeout 600;
        proxy_send_timeout 600;
        proxy_read_timeout 600;
    }
}

Проверка и запуск на сервере
nginx -t
systemctl reload nginx
systemctl stop nginx

2. Запускаем свое приложение на своем компьютере (localhost)
python run.py (к примеру, запуск бэкенда)
ssh -R 8000:127.0.0.1:8000 root@____ (логин@IP нашего сервера, обычно root - логин по умолчанию)
После потребуется ввести пароль подкобчения к серверу
8000 - порт запуска

npm run dev (к примеру, запуск фронтенда
ssh -R 3000:localhost:3000 root@____ (логин@IP нашего сервера, обычно root - логин по умолчанию)
После потребуется ввести пароль подкобчения к серверу
3000 - порт запуска
